# Proyecto-1-EDD
Analisis de Redes Sociales
Este proyecto se basa en la visualización y funcionamiento de una red social donde se agregan y eliminan usuarios, además de crearse relaciones entre ellos, que se verán reflejadas en un grafo. Se implementaron diversas estructuras de datos para la realización del proyecto entre otros recursos adicionales para hacerlo más creativo.
